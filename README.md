# node-auth
Authentication system using Vanilla JS, Node and HTML


## Tools used
- Fastify (Web Application Server)
- MongoDB (Database)


## Features
- Login, Sign Up, and User Registration
- Password Reset
- 2 Factor Authentication
- Email Verification
- User Logout
